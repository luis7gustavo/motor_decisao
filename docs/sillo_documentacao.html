Decisao

O motor separa oportunidade real de ruido. Compra teste exige margem, demanda, match forte, fontes suficientes e ausencia de flags bloqueantes.

Operacao

Comandos principais para operar a base local, coletar fornecedores, rodar o motor e consultar a API.

Ambiente

.\SetupMotor.cmd
Invoke-RestMethod http://127.0.0.1:8010/health

Fornecedores

docker compose exec -T api python scripts/collect_suppliers.py --supplier mirao
docker compose exec -T api python scripts/build_decision_engine.py --import-megamix

Resumo

Invoke-RestMethod http://127.0.0.1:8010/decision-engine/summary

Oportunidades

Invoke-RestMethod "http://127.0.0.1:8010/decision-engine/opportunities?recommendation=comprar_teste"

Guardrails

O motor e conservador por desenho. Uma oportunidade so vira compra teste se passar por qualidade de match, margem, demanda e cobertura de fontes.

Validar equivalencia

Confirmar se o produto do fornecedor e realmente o mesmo produto observado no mercado.

Comprar pequeno

A primeira acao recomendada e teste controlado, nao reposicao grande.

Aprender com o giro

Registrar venda, devolucao, margem real e tempo de estoque para treinar o proximo ciclo.
